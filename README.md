# What’s wrong with my plot?
## What this resource is:
A beginner’s guide to what can go wrong when visualising data using ggplot2 in R. It covers some of the most common reasons why a ggplot2 plot looks different or less informative from what you expected e.g. wrong number of groups appearing, unexpected ordering, missing lines, collapsed categories etc.

## What this resource is not:
*Styling and appearance* - colours, backgrounds, gridlines, fonts, themes, or formatting of axis labels, titles, captions, and annotations.  

*Choosing the right plot for your data* - there are already excellent resources to guide you on this.
